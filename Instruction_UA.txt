Second Army Checker (SAC) — Інструкція користувача
---------------------------------------------------

Програма призначена для перевірки та оновлення складу клану "Second Army" в грі. Вона працює з двома CSV-файлами: input.csv (дані від гравців) та output.csv (збережені результати).

Файли:
1. SAC.exe           — виконуваний файл програми для Windows.
2. Test/input.csv     — вхідні дані (нікнейм гравця та очки).
3. Test/output.csv    — вихідні дані (оновлюються програмою).
4. Converter/         — директорія для завантаження даних в форматі скріншотів.
5. Instruction_UA.txt — ця інструкція.

Запуск програми:
----------------
1. Двічі клікнути на SAC.exe або запустити через командний рядок.
2. На екрані з'явиться меню з можливими командами:

   N або New або 0  — створити новий файл output.csv з порожніми даними.
   C або Check або 1 — перевірити файли на помилки (наявність, правильність даних).
   S або Start або 2 — обробити input.csv та оновити output.csv.
   V або Visualize або 3 — відобразити поточний результат в файлі output.csv.
   O або OCR або 4 — зчитати дані з папки Converter для запису в input.csv.
   E або Exit або 5  — вийти з програми.

Кроки запуску:
---------------
1. Запустити SAC.exe 
2. Завантажити файл input.csv і розмістити в тій самій директорії що і SAC.exe
3. Запустити Check - ввести 1, щоб перевірити готовність до роботи.
4. Якщо всі перевірки будуть True, запустити Start - ввести 2, щоб виконати обробку даних.
5. Скопіювати результати, записати.
6. Закрити програму через Exit - ввести 5.

ВАЖЛИВО!!!
Якщо output.csv - не існує потрібно ввести 0, щоб згенерувати її.

Рекомендації:
-------------
- Можете використовувати файли з папки /Test якщо перемістите їх в корінь програми.
- Завжди оновлюйте input.csv перед запуском Start для актуальних даних.
- Ви можете оновити свій input.csv завантаживши картинки з результатами в папку Converter, після чого використати опцію OCR (4).
- Використовуйте тільки SAC.exe для запуску на Windows.
